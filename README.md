# notes

Note taking app written in rust.

```
notes 

USAGE:
    notes [SUBCOMMAND]

OPTIONS:
    -h, --help    Print help information

SUBCOMMANDS:
    add       add a new note.
    delete    delete existing note.
    get       read note.
    help      Print this message or the help of the given subcommand(s)
    update    replace note text.
```
